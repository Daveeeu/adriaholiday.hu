# Adria Holiday - Global Design Refinement

## Overview
Premium Mediterranean travel brand design system - Editorial magazine aesthetic with cinematic storytelling.

## Core Design Principles

### 1. **Editorial Magazine Layout**
- Full-bleed imagery with editorial overlays
- Asymmetric layouts for visual interest
- More breathing space between sections
- Text-on-image with premium gradients
- Magazine-style typography hierarchy

### 2. **Premium Spacing System**
- Section padding: `py-40` (instead of py-24/py-32)
- Container max-width: `1500px` (more spacious)
- Grid gaps: `gap-8` to `gap-24` (generous spacing)
- Padding: `px-20` on desktop (wider margins)

### 3. **Immersive Imagery**
- Full-bleed hero images (-ml-20 technique)
- Larger image scales (700px+ heights)
- Slow ken-burns animations (30s duration)
- Cinematic gradient overlays
- Text integrated into images

### 4. **Emotional Storytelling**
- Customer quotes with glassmorphism cards
- Editorial text overlays on images
- Location badges (e.g., "Dalmácia • Horvátország")
- Aspirational headlines with gradients
- Personal testimonials

### 5. **Less Card-Heavy Design**
- Reduced border-radius usage where appropriate
- More borderless content sections
- Glassmorphism instead of solid cards
- Floating quote cards sparingly
- Editorial blocks instead of grid cards

### 6. **Typography Hierarchy**
```
Headlines: clamp(2.25rem, 5vw, 3.5rem), weight 700
Subheadlines: 1.125rem-1.25rem, weight 500-600
Body: 0.9375rem-1rem, line-height 1.6-1.8
Labels: 0.875rem uppercase, tracking 0.05-0.1em
```

### 7. **Color Application**
- Primary: #00c389 (turquoise)
- Gradient: #00c389 → #16b8ff
- Text: Gray-900 (not black)
- Backgrounds: White → #f8fafc gradient
- Accents: Minimal, purposeful

### 8. **Motion Design**
- Slower animations (1-1.4s)
- Cubic-bezier easing [0.16, 1, 0.3, 1]
- Ken-burns on images (30s linear)
- Subtle parallax (optional)
- Staggered reveals (0.2-0.3s delays)

- Premium glassmorphism cards

### **Hero Section**
- Large atmospheric globe (780px)
- Premium glassmorphism badges
- Grid: 1.2fr : 0.8fr ratio

## What to Avoid
- ❌ Gaming/neon aesthetics
- ❌ Excessive glassmorphism
- ❌ Too many floating badges
- ❌ Generic card grids
- ❌ SaaS dashboard feeling
- ❌ Overcrowded layouts
- ❌ Stock photo feeling
