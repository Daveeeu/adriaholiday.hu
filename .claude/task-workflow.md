# Kötelező workflow

Minden feladat előtt:

1. Olvasd el a releváns .claude dokumentumokat.
2. Auditáld a jelenlegi implementációt.
3. Keress hasonló megoldásokat a projektben.
4. Ne duplikálj kódot.
5. Használd a meglévő komponenseket.
6. Gondolkodj rendszer szinten.

Munka közben:

- Ne törj el működő funkciót.
- Tartsd be az architecture szabályokat.
- Legyen production-ready.
- Kerüld a hardcode-ot.
- Legyen bővíthető.

Munka végén:

- Self review.
- Build.
- Tesztek.
- Git status.
- Commit.
- Push.

A válasz mindig tartalmazza:

- Audit
- Döntések
- Módosított fájlok
- Kockázatok
- Build eredmény
- Teszt eredmény
- Commit hash
